## Lab One
Python script to create a **database**

## Run Locally

Clone the project

```bash
  git clone https://github.com/PLP-Database-DEPT/hands-on-example/
```

Go to the project directory

```bash
  cd week-1/lab/lab-1
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Create Database

```bash
  python main.py
```
